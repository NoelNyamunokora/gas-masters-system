# Static Files Directory

## Logo Placement

Please place your `logo.png` file in this directory for it to appear in the application.

The logo should be:
- Named exactly `logo.png`
- Preferably square or rectangular format
- Recommended size: 200x200 pixels or similar
- PNG format with transparent background (optional)

The logo will appear:
- On the login screen
- On the registration screen  
- In the navigation bar (when logged in)
- As the browser favicon
